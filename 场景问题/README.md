# 场景问题

## 说明

这个文件夹专门存放**场景类面试问题**的回答。与单知识点文档不同，场景问题的回答会**串联多个分类的知识点**，把分散的知识织成一条完整的答题线。

## 回答格式

每个场景问题文档包含以下板块：

- **场景描述** — 还原面试官的提问
- **答题思路** — 先讲思考框架，再逐层展开
- **完整回答** — 从背景 → 分析 → 方案 → 优化，给出可以直接口述的答案
- **知识点串联** — 列出本题涉及的所有已有文档，形成知识脉络
- **追问预判** — 面试官可能的后续追问
