# Tech Knowledge Base

计算机专业面试知识库。通过提问逐步积累面试知识点，所有回答以 Markdown 保存，按分类组织，文档间通过链接互相关联。

## 目录结构

```
tech-knowledge-base/
├── README.md                 # 本文件
├── TEMPLATE.md               # 回答模板（修改此文件可调整回答格式）
├── INDEX.md                  # 知识库总索引
├── 计算机网络/                # 网络协议、TCP/IP、HTTP 等
├── 操作系统/                  # 进程、线程、内存管理、文件系统等
├── 数据结构与算法/            # 排序、树、图、动态规划等
├── 数据库/                    # SQL、事务、索引、分布式数据库等
├── 大模型-AI/                 # Transformer、LLM 原理、Prompt 等
├── 系统设计/                  # 分布式系统、高可用、架构设计等
├── 编程语言/                  # 语言特性、编译原理等
└── 场景问题/                  # 场景类面试题，串联多分类知识点
```

## 使用方式

1. 直接提问面试问题
2. 回答会按照 `TEMPLATE.md` 中的模板格式生成
3. 如果问题已存在，会在原有文档上补充完善
4. 每次新增内容都会自动更新 `INDEX.md` 和对应分类的索引

## 回答模板

所有回答遵循 [TEMPLATE.md](TEMPLATE.md) 中定义的格式，包含四个板块：

- **面试精答** — 简洁版，适合口述
- **原理深入详解** — 详细版，深入底层原理
- **常见追问与应对** — 面试官可能的追问方向
- **知识关联** — 链接到相关知识点

修改 `TEMPLATE.md` 即可调整后续所有回答的风格和结构。
